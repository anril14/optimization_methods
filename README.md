Файлы с "test_ ..." в названии - сохранения предыдущих решений и не во всех случаях верные

ЛАБОРАТОРНАЯ РАБОТА № 3. ЧИСЛЕННЫЕ МЕТОДЫ НУЛЕВОГО ПОРЯДКА РЕШЕНИЯ ЗАДАЧ НА БЕЗУСЛОВНЫЙ ЭКСТРЕМУМ - 3_hooke_jeeves.py (Метод хука-дживса)
ЛАБОРАТОРНАЯ РАБОТА 4. ЧИСЛЕННЫЕ МЕТОДЫ ПЕРВОГО ПОРЯДКА РЕШЕНИЯ ЗАДАЧ НА БЕЗУСЛОВНЫЙ ЭКСТРЕМУМ - 4_fastest_descent.py (Метод наискорейшего спуска)
ЛАБОРАТОРНАЯ РАБОТА 5. ЧИСЛЕННЫЕ МЕТОДЫ РЕШЕНИЯ ЭКСТРЕМАЛЬНЫХ ЗАДАЧ С ОГРАНИЧЕНИЯМИ - 5_penalty_sd.py (Метод штрафных функций)
ЛАБОРАТОРНАЯ РАБОТА 6. МЕТОД ДИНАМИЧЕСКОГО ПРОГРАММИРОВАНИЯ - 6_backpack_with_way.py (Метод динамического программирования - задача о рюкзаке)
ЛАБОРАТОРНАЯ РАБОТА 7. ЖАДНЫЕ АЛГОРИТМЫ - 7_greedy_alg_optimal.py (Задача о приоритете мероприятий)
ЛАБОРАТОРНАЯ РАБОТА 8. ЭВРИСТИЧЕСКИЕ АЛГОРИТМЫ - 8_ant_colony_asymm.py (Муравьиный алгоритм)
